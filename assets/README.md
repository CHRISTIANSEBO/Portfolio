# Project media

Drop project screenshots or short demo GIFs here, then they appear
automatically on the matching project card.

Expected filenames (referenced by `index.html`):

| File | Project |
|------|---------|
| `research-assistant.png` (or `.gif`) | AI Research Assistant |
| `jean.png` (or `.gif`) | AI Email Assistant — "Jean" |

Tips:
- Aim for a 16:9 ratio (e.g. 1280×720) so it fills the card media area cleanly.
- A 15–30s GIF/MP4-as-GIF of the agent working converts far better than a static shot.
- Keep files reasonably small (< ~5 MB) so the page stays fast.

If a file isn't present yet, the card shows a tasteful "preview coming soon"
placeholder instead of a broken image.

Allowed extensions served in production: `.png .jpg .jpeg .gif .webp .svg`
